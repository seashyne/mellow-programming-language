# Mellow v2.9.5 Development Benchmark

Source commit: `8d278e295cfbed8752f0ecb3fed71d7c7a2814d4`

The branch is `dev/2.9.5`. Package metadata still declares `2.9.4`, so these
numbers describe the v2.9.5 development runtime, not a released v2.9.5 build.

## Standalone C Runtime

The standalone workload sums integers from 0 through 49,999. Each result is the
median of seven runs after one warm-up run.

| Target | Execution | Median | p95 | Result |
| --- | --- | ---: | ---: | ---: |
| x86_64 Linux | native GitHub runner | 9.318 ms | 9.406 ms | 1,249,975,000 |
| ARM64 Linux | QEMU on x86_64 runner | 128.181 ms | 128.814 ms | 1,249,975,000 |

The ARM64 number includes QEMU translation overhead. It proves that the ARM64
binary builds, executes correctly, and can be regression-tested, but it is not a
measurement of real ARM64 hardware performance.

For reference, the same x86_64 workload on the local Windows Release build had
a median of 10.707 ms across 11 runs.

## Native Extension Runtime

Local Windows x86_64, Python 3.11, seven runs per workload:

| Workload | Reference VM | Native C | Native speedup |
| --- | ---: | ---: | ---: |
| Arithmetic loop | 75,214 iterations/s | 2,105,684 iterations/s | 28.00x |
| Function calls | 51,969 calls/s | 877,020 calls/s | 16.88x |
| Money addition | 48,929 operations/s | 149,886 operations/s | 3.06x |
| Ledger posting | 3,757 transactions/s | 3,224 transactions/s | 0.86x |
| Data transforms | 2,477,551 rows/s | 8,201,965 rows/s | 3.31x |

Ledger posting remains host-service-heavy and is approximately 14% slower
through the native bridge. This is the clearest optimization target found by
this run.

## Reproduction

Standalone results are generated by `benchmarks/standalone_runtime_benchmark.py`
and uploaded by GitHub Actions as JSON artifacts named
`mellow-native-x64-benchmark` and `mellow-arm64-qemu-benchmark`.

The complete workflow passed native x64, ARM64, tooling reference, and package
release gates:

https://github.com/seashyne/mellow-programming-language/actions/runs/27903651255
