# CLI 1.6.0

## New commands

```powershell
mellow add @mellow/core-ai --project-dir .
mellow remove @mellow/core-ai --project-dir .
mellow run app/main.mellow --registry https://registry.example.com
mellow doctor
```

## `mellow add`
Adds a dependency to `mellow.toml` and installs it recursively.

## `mellow remove`
Removes the dependency from the manifest, lockfile, and installed package store.

## `mellow run`
Before execution, Mellow now tries to resolve imports in the project root and writes `.mellow_runtime.json`. Use `--no-resolve` to disable.
