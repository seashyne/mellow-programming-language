# MellowLang v1.5.1

## Highlights

1. Package manager
   - `mellow pkg init`
   - `mellow pkg publish`
   - `mellow pkg install`
   - `mellow pkg build`

2. Game scripting engine
   - `game.scene_create()`
   - `game.entity_create()`
   - `game.entity_move()`
   - `game.collide_aabb()`
   - `game.anim_frame()`

3. AI API
   - `ai.api_providers()`
   - `ai.api_configure()`
   - `ai.api_complete()`
   - `ai.api_embed()`

4. Compiler
   - `mellow compile app.mellow --target bytecode`
   - `mellow compile app.mellow --target python`

5. Documentation
   - Added overview, examples, and workflow guidance for package/game/AI/compiler usage.
