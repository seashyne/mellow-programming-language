# Mellow v2.2.0 MMG Render Core Pack

## Added

- MMG scene graph parsing
- sprite and texture metadata
- camera support
- frame-loop support
- input event wiring
- render graph metadata
- new example: `examples/mmg_render_core_demo.mel`
- updated `core-mmg` starter package surface

## Notes

This pack focuses on MMG render/runtime structure and cross-platform behavior. It is still canvas-backed and does not claim GPU-native parity with OpenGL.
